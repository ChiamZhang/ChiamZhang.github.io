前缀有哪些趋势

表示上下的前缀

  1. ultra 在上面 超过 ultraviolet UV 紫外线 violet 紫罗兰色

  2. infra 在下面 低于 infrared 红外线

  3. hyper 在上面 hypersensitive

  4. hypo 在下面 hypothesis

  5. 简单的一些 up-down over-under

    downplay=underestimate 轻视 低估

    downturn = decline 下降趋势

更常见的下sub de

  1. subway 地铁

  2. suburb 郊区 sub+urban

  3. subsitute 代替 sub+ stitute =repace

  4. subsequent 后来的 sub+seq + uent

    sequence 顺序 consequence 结果

  5. substantial 根本上 sub+sand+ial

  6. submit 提交

  7. submissive 服从的

  8. emit 发送 transmit 传输

  9. subordinate 从属

  10. sub-discipline 二级学科

sub 的变形

sub 遇到比sub中的b还弱的辅音，将b改成与此字母。

  1. successive 连续的，继承的
  2. suffrer 遭受，承受 undertake undergo
  3. supplement 补充 supplementary=extra
  4. sustain 维持 sustainable 可持续的
其他前缀的各种变形

  1. con家族：co com col cor

  2. in家族：im il ir

  3. syn家族：sym syl

  4. dis家族：di dif

  5. ob家族: of op

    ob=against obstacle障碍 offend 冒犯 opponent 对手 opposite 相反的，对面的

de 向下变坏
  1. decrease
  2. decline
  3. descend
  4. destruct
  5. deforestation
  6. degradation
  7. deteriorate
  8. detrimental
  9. deprive

不属于de词根的：

  1. decorate
  2. destiny
  3. delicate
  4. democratic

一旦读成长音[di:] ：解开

  1. decode
  2. detoxify
  3. defoliate defoliation

表示前后的前缀

ante 在前面

  1. antecedent
  2. ancestor
  3. antique

fore 前面的

  1. forefather
  2. forecast

post 在后边

  1. postwar
  2. postpone
  3. postgraduate

pre 前 - re 后

  1. preview 预习 review 复习
  2. respond 反应
  3. reflect 反射
  4. retract 收缩
  5. reject 拒绝
  6. reduce 减少

re也有重复的意思：

  1. replay
  2. republish
  3. reproduce
  4. restrict ≠ strict
  5. reluctant

区别：

re+词根=反向趋势

re+实义词=重复

pre 在…之前 pro 向前

  1. prehistoric 史前的

  2. prehistory 史前

  3. presume 推测 ≈ assume

  4. precaution 预防 警惕

    依据真题补充：

    5. counter-strike 反对细节

  6. prevalent 流行的

  7. predecessor 祖先

    依据真题补充：

  8. prestige 威望，名声

  9. profound
