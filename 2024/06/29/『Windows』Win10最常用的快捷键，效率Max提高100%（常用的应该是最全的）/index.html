『Windows』Win10最常用的快捷键，效率Max提高100%（常用的应该是最全的）

『Windows』Win10 最常用的快捷键，效率 Max 提高 100%（常用的应该是最全的）

写在最前面

这是博主爆肝了一晚上给写出来，因为很多博客和资料中仍然使用的 xp win7 和 win8
的快捷键，我不断地的实验和尝试，总结出以下的快捷键，希望可以帮助到你。
最后，==转载请注明出处==

Win10 窗口快捷键`：

  1. Win + ←：最大化窗口到左侧的屏幕上（与开始屏幕应用无关）
  2. Win + →：最大化窗口到右侧的屏幕上（与开始屏幕应用无关）
  1. Win+ ↑：最大化窗口（与开始屏幕应用无关）
  2. Win+ ↓：最小化窗口（与开始屏幕应用无关）
  1. Win+ SHIFT +↑：垂直拉伸窗口，宽度不变（与开始屏幕应用无关）
  2. Win+ SHIFT +↓：垂直缩小窗口，宽度不变（与开始屏幕应用无关）
  1. Win+SHIFT+←：将活动窗口移至左侧显示器 （与开始屏幕应用无关）
  2. Win+SHIFT+→：将活动窗口移至右侧显示器（与开始屏幕应用无关）
  1. Alt + Tab:切换窗口,不是新的，但任务切换界面改进

  1. Win + Tab（松开键盘界面不会消失）： 任务视图

  2. Win + Ctrl + D： 创建新的虚拟桌面

  3. Win + Ctrl + F4：关闭当前虚拟桌面

  4. Win + Ctrl + 左/右：切换虚拟桌面

Win 键组合快捷键

  1. Win+A：接受所有来电 （在 microsoft Lync 中）
  2. Win+B：光标移至通知区域
  3. Win+D：显示桌面，第二次键击恢复桌面 （不恢复开始屏幕应用）
  4. Win+E：资源管理器
  5. Win+F：反馈（没用）
  6. Win+G：Xbox Gamebar（可以提供游戏相关功能）
  7. Win+H：听写功能（没啥用）
  8. Win+I：打开`Win10 设置
  9. Win+K：打开连接设备
  10. Win+L：锁屏
  11. Win+M： 最小化所有窗口
  12. Win+ P：演示设置
  13. Win+Q： 搜索功能
  14. Win+R：运行
  15. Win+S=Win+Q`： 搜索功能
  16. Win+T：切换任务栏上的程序
  17. Win+U：显示`Win10 设置
  18. Win+V：系统粘贴板
    强烈推荐，非常好用
  19. Win+W：白板和截图
  20. Win+X：相当于右键点击 Win 标，应该叫移动控制中心
  1. Win+ Home：最小化所有窗口，第二次键击恢复窗口（不恢复开始屏幕应用）
  2. Win+pause：显示“系统属性”对话框
  3. Win+Ctrl+F：搜索计算机（如果你在网络上）
  4. Win+Shift+M：在桌面恢复所有最小化窗口（不恢复开始屏幕应用）
  5. Win+F1：打开 Windows 帮助和支持（没用）
  1. Win+减号：缩小（放大镜）
  2. Win+加号：放大（放大镜）
  3. Win+Esc：关闭放大镜
  1. Win+空格键：切换输入语言和键盘布局
  2. Win+，：临时查看桌面

  1. Win+Tab：循环切换应用

  2. Win+Shift+Tab：反向循环切换应用

  3. Win+Ctrl+Tab：循环切换应用，切换时手动选择应用

  4. Win+/：恢复默认输入法

  1. Win+ 数字键：打开或切换位于任务栏指定位置的程序
  2. Shift + Win键 + 数字 启动锁定到任务栏中的由该数字所表示位置处的程序的新实例
  3. Ctrl + Win键 + 数字 切换到锁定到任务栏中的由该数字所表示位置处的程序的最后一个活动窗口
  4. Alt + Win键 + 数字 打开锁定到任务栏中的由该数字所表示位置处的程序的跳转列表 (Jump List)
  5. Ctrl + Win键 + Tab 通过 Aero Flip 3-D 使用箭头键循环切换任务栏上的程序
  6. Ctrl + Win键 + B 切换到在通知区域中显示消息的程序

通用的快捷键

  1. Ctrl+A ：全选
  2. Ctrl+C(或 Ctrl+Insert)： 复制选择的项目
  3. Ctrl+X： 剪切选择的项目
  4. Ctrl+V(或 Shift+Insert) ：粘贴选择的项目
  5. Ctrl+Z： 撤消操作
  6. Ctrl+Y ：重新执行某项操作
  7. Alt+Enter ：显示所选项的属性
  8. Alt+F4 ：关闭活动项目或者退出活动程序
  9. Alt+空格键： 为活动窗口打开快捷方式菜单
  10. Ctrl+F4 ：关闭活动文档(在允许同时打开多个文档的程序中)
  11. Ctrl+Alt+Tab： 使用箭头键在打开的项目之间切换
  12. Ctrl+鼠标滚轮： 更改桌面上的图标大小
