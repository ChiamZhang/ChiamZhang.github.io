『Visual Studio Code』解决 Visual Studio (VS) 插件下载缓慢

1.关闭 IPV6 协议

因为如果都支持 IPV6 协议，会自动使用 IPV6 下载扩展。
因为 IPV6 还没有建立完善，所以可能会比较慢。
百度经验:怎样关闭 IPV6 协议

2.给 IPV6 添加 DNS

百度： 2400:da00::6666
谷歌： 2001:4860:4860::8888

为什么要添加两个？
因为也要保证国内的网络可以正常访问。

参考设置 IPV4 DNS

首先 Win+R 打开 cmd
