『开发』解决打开虚拟机 VMware Workstation 报错无法改变虚拟机的电源状态 Operation inconsistent with current state 问题

原因

不正常的关闭虚拟机导致，如没有关闭虚拟机，就关机重启等原因。
在你的本次会话中，你的虚拟机是关闭的，但是 VM 的服务中他还是开启的。
大概是这个原因，按照网上的方式重启虚拟机服务也可以解决。

解决方式

简单的解决方式，就是使用管理员运行 VMware 即可。

注意

以后记得要先挂起虚拟机或者关闭虚拟机之后再关闭 VMware，然后在关机。
强制关闭 VM 也可能出现这种问题。
