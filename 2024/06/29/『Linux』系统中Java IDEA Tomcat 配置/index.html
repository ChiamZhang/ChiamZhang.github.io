『Linux』系统中Java IDEA Tomcat 配置

『Linux』系统中 Java IDEA Tomcat 配置

环境搭建记录-开发环境搭建：

  1. IDEA 安装
    https://www.jetbrains.com/idea/download/#section=linux

解压后执行 bin 目录 idea.sh 运行

  1. JDK 安装
  1. 下载 jdk15

官方网站:https://www.oracle.com/java/
下载页面:https://www.oracle.com/cn/java/technologies/javase-downloads.html 2) 安装 jdk15

解压安装文件的压缩包：
tar -zxvf jdk-15_linux-x64_bin.tar.gz

把解压后的 jdk 目录，移动到安装目录/usr/local/soft 目录下:
mv jdk-15 /usr/local/soft/

  1. 配置 java 的环境变量:

编辑 profile 文件,把命令保存到文件中
vi /etc/profile
内容:

export JAVA_HOME=/usr/local/soft/jdk-15
export CLASSPATH=.:$JAVA_HOME/lib
export PATH=$PATH:$JAVA_HOME/bin

使变量生效
···· source /etc/profile

  1. 检查 java 安装是否完成
    java –version
  1. Tomcat9 安装与部署
  1. /usr/local/tomcat，创建好文件夹、目录。
  2. 执行命令下载 Tomcat9 文件 约 1 小时
    sudo wget https://dlcdn.apache.org/tomcat/tomcat-9/v9.0.58/bin/apache-tomcat-9.0.58.tar.gz
    换用国内源 https://mirrors.cnnic.cn/apache/tomcat/tomcat-9/v9.0.58/bin/apache-tomcat-9.0.58.tar.gz
    3） 安装 sudo tar -zxvf ‘/usr/local/tomcat/apache-tomcat-9.0.58.tar.gz‘
    4） 修改 Tomcat 文件夹权限
    sudo chown -R 用户名:用户名 apache-tomcat-9.0.58
  1. IDEA 整合 Tomcat

选择 Tomcat 并选择解压安装的目录即可。
